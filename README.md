# Indoor-Jungle
🌿 Indoor Jungle  Indoor Jungle is a fully responsive indoor plant selling website built using HTML and Tailwind CSS. The website showcases a curated collection of indoor plants, stylish pots, and plant accessories designed to enhance modern living spaces. It features a clean layout, modern UI, and a smooth browsing experience across all devices.

🚀 Features

🌱 Beautiful indoor plant product sections

🪴 Stylish pots and plant accessories display

📱 Fully responsive design (Mobile, Tablet & Desktop)

🎨 Modern and clean user interface

⚡ Utility-first styling with Tailwind CSS

🛠️ Technologies Used

HTML5

Tailwind CSS

Responsive Web Design

📱 Responsive Design

The website is designed to work seamlessly across all screen sizes using Tailwind CSS responsive utilities. Layouts automatically adjust for mobile, tablet, and desktop devices.

🎯 Project Goal

This project was created to practice front-end development skills, improve responsive design techniques, and build a modern e-commerce style website using Tailwind CSS.
